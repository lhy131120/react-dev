## 動畫效果

### 1. 卡片淡入
```
時間線: 0s → 0.8s
初始: opacity: 0, translateY(30px)
最終: opacity: 1, translateY(0)
效果: 卡片從下方淡入進場
```

### 2. Hover 上升
```
時間線: 0.3s
效果: 
  - 向上浮起 8px
  - 陰影變深 25px
  - 邊框顏色變深
```

### 3. 浮動粒子
```
粒子大小: 小、中、大三種
顏色: 橘色、綠色、棕色
動畫: 8-10s 來回浮動
方向: 向上，帶橫向偏移
```

### 4. 導航連結下劃線
```
初始: width: 0
Hover: width: 100%
時間: 0.3s
效果: 漸層色線條從左向右展開
```

### 5. 按鈕白色流動
```
偽元素: ::before
初始: left: -100%
Hover: left: 100%
時間: 0.3s
效果: 白色光束從左滑過按鈕
```

## 特殊效果

### 毛玻璃 (Glassmorphism)
```
效果: backdrop-filter: blur(10px)
應用於:
  - 卡片背景
  - 導航列
  - 模態對話框
  - 分頁容器
特點: 透明但模糊的背景
```

### 漸層邊框
```
technique:
  background: linear-gradient(white, white) padding-box, 
              linear-gradient(90deg, #d97706, #f59e0b) border-box;
  border: 2px solid transparent;

應用於:
  - 購物車數量選擇器
  - 某些高級卡片
```

### 陰影層次
```
普通卡片:
  box-shadow: 0 10px 40px rgba(217, 119, 6, 0.1),
              inset 0 1px 0 rgba(255, 255, 255, 0.6);

Hover 卡片:
  box-shadow: 0 25px 50px rgba(217, 119, 6, 0.2),
              inset 0 1px 0 rgba(255, 255, 255, 0.8);

特點: 多層陰影創造深度感
```
